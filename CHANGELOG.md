## [Unreleased]

### Added
- Data structures: Stack, Queue, LinkedList, BinarySearchTree, MinHeap.
- Algorithms: bubble/insertion/merge/quick sort, linear/binary search,
  BFS/DFS traversal, memoized Fibonacci, longest common subsequence.
- String helpers: palindrome, anagram, and word-reversal checks.
- Maths helpers: gcd, lcm, factorial, sieve of Eratosthenes.
- pytest suite and GitHub Actions CI.
